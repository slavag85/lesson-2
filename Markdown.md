## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочками (*).

Например,вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумировать.

Например, вот так:
1. Первый пункт

2. Второй пункт

3. Третий пункт
